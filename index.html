ISOBlue 2.0

ISOBlue 2.0 is OATS group's latest development for collecting machine data as well as connecting ag machinery to the Cloud. Using Toradex's iMX6 platform and open source tools like Apache Kafka we are able to stream process the collected data in real-time.
