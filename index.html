ISOBlue

Open-Source Agricultural Machinery Sensor Hub

Get Started

Easy to Deploy

Start collecting CAN and GPS data from your machine's ISOBUS diagnostic port using an ISOBlue. Just plug it in. Yes, as easy as that.

Ready to Expand

A multimodal dataset is a gateway to learn more about machine operation and farming logistics. ISOBlue has a number of peripherals for connecting additional sensors.

Driven by Community

ISOBlue builds on Free and Open Source Software (FOSS). You have the freedom to modify and make it better for your need.
